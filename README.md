## 📘 课程简介

计算机图形学是一门研究如何使用计算机生成、处理和显示图形的基础课程。
本课程主要内容包括：

* 三维几何表示与变换
* 渲染管线（Rendering Pipeline）
* 投影变换（Projection）
* 光照与着色（Shading）
* 图形交互与动画

通过理论学习与编程实验相结合，加深对图形学核心算法与实现机制的理解。

---

## 🧪 仓库简介

本仓库用于存放《计算机图形学》课程的实验代码与相关文档。

所有实验基于 **Python + Taichi** 实现，涵盖从基础图形绘制到三维变换与交互的多个内容。
